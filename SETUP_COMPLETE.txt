📞 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Issue: Cannot connect to database
Solution:
  1. Ensure MySQL server is running
  2. Check credentials in DBConnection.java
  3. Run database-setup.sql
  4. Verify database exists

Issue: 404 Page Not Found
Solution:
  1. Verify servlet mapping in web.xml
  2. Check URL spelling
  3. Rebuild WAR file
  4. Restart Tomcat

Issue: Table not found
Solution:
  1. Run: mysql -u root -p < database-setup.sql
  2. Verify database name: studentdb
  3. Check table: USE studentdb; SHOW TABLES;
